# buy-event
How to run?
1. Set values to environment variables:
```bash
MAIN_PHONE=source_phone_number
TWILIO_ACCOUNT_SID=twilio_sid
TWILIO_AUTH_TOKEN=twilio_auth_token
MAIN_EMAIL=source_email
MAIN_EMAIL_PASSWORD=source_email_password
```
2.
```go
go run main.go
```
or
```go
go build
./buy-event
```
